## Modern Sans-Serif Pairings

Clean, professional, versatile. Suitable for most web applications.

### 1. Inter + Inter
- **Heading:** Inter (600-700)
- **Body:** Inter (400)
- **Style:** Neutral, highly legible, variable font
- **Best for:** SaaS, dashboards, developer tools, any UI-heavy product

## Modular Scale Ratios

Choose a ratio based on the content type and visual density needed.

| Ratio | Name | Value | Best For |
|-------|------|-------|----------|
| 1.067 | Minor Second | tight | Dense data UIs, compact dashboards |
| 1.125 | Major Second | compact | Admin panels, form-heavy interfaces |
| 1.200 | Minor Third | balanced-compact | Dashboards, data-dense SaaS products |
| 1.250 | Major Third | balanced | Most web apps, general purpose (DEFAULT) |
| 1.333 | Perfect Fourth | expressive | Marketing sites, editorial, presentations |
| 1.414 | Augmented Fourth | dramatic | Landing pages, brand storytelling |
| 1.500 | Perfect Fifth | bold | Hero sections, luxury, large-format displays |
| 1.618 | Golden Ratio | grand | Art, fashion, ultra-premium branding |

### Generating a Scale

From a base size of 16px with Major Third (1.250):

```
Step -2: 16 / 1.250^2 = 10.24px  → 0.64rem  (caption, fine print)
Step -1: 16 / 1.250   = 12.80px  → 0.80rem  (small, labels)
Step  0: 16.00px       = 16.00px  → 1.00rem  (body text -- BASE)
Step  1: 16 * 1.250    = 20.00px  → 1.25rem  (large body, lead)
Step  2: 16 * 1.250^2  = 25.00px  → 1.563rem (h4)
Step  3: 16 * 1.250^3  = 31.25px  → 1.953rem (h3)
Step  4: 16 * 1.250^4  = 39.06px  → 2.441rem (h2)
Step  5: 16 * 1.250^5  = 48.83px  → 3.052rem (h1)
Step  6: 16 * 1.250^6  = 61.04px  → 3.815rem (display)
```

## Responsive Typography Patterns

### Fluid Typography with clamp()

Use `clamp(min, preferred, max)` for smooth scaling between breakpoints.

```css
:root {
  /* Base: scales from 15px at 320px viewport to 18px at 1280px */
  --text-base: clamp(0.938rem, 0.875rem + 0.3125vw, 1.125rem);

  /* Headings: scale more aggressively */
  --text-lg:  clamp(1.125rem, 1.0rem + 0.625vw, 1.5rem);
  --text-xl:  clamp(1.35rem, 1.1rem + 1.25vw, 2rem);
  --text-2xl: clamp(1.6rem, 1.2rem + 2vw, 2.75rem);
  --text-3xl: clamp(1.9rem, 1.3rem + 3vw, 3.75rem);
  --text-4xl: clamp(2.25rem, 1.25rem + 5vw, 5rem);
}
```

### Breakpoint-Based Alternative

When `clamp()` is not suitable (older browser support):

```css
/* Mobile first */
:root {
  --text-base: 1rem;
  --text-xl: 1.5rem;
  --text-3xl: 2rem;
}

/* Tablet (768px+) */
@media (min-width: 48rem) {
  :root {
    --text-base: 1.0625rem;
    --text-xl: 1.75rem;
    --text-3xl: 2.5rem;
  }
}

/* Desktop (1280px+) */
@media (min-width: 80rem) {
  :root {
    --text-base: 1.125rem;
    --text-xl: 2rem;
    --text-3xl: 3.25rem;
  }
}
```

## Line Height and Spacing

Line heights decrease as font size increases:

| Text Size | Line Height | Rationale |
|-----------|-------------|-----------|
| xs-sm (< 14px) | 1.6 - 1.7 | Small text needs more vertical room |
| base (14-18px) | 1.5 - 1.6 | Optimal reading line height |
| lg (18-24px) | 1.4 - 1.5 | Slightly tighter for larger body |
| xl-2xl (24-40px) | 1.2 - 1.3 | Headings need tighter leading |
| 3xl+ (40px+) | 1.05 - 1.15 | Display type needs minimal leading |

**Letter spacing guidance:**

| Context | Letter Spacing | Example |
|---------|---------------|---------|
| Body text | 0 (normal) | Default, do not adjust |
| All-caps labels | 0.05em - 0.1em | `SECTION TITLE` |
| Large display headings | -0.02em - -0.03em | Tighten for optical balance |
| Tight headings (geometric fonts) | -0.01em - -0.025em | Inter, Outfit at large sizes |
| Monospace | 0 | Never adjust monospace spacing |

**Paragraph spacing:**

```css
p {
  margin-bottom: 1em; /* Equal to current font size */
}

/* Tighter for UI contexts */
.ui-text p {
  margin-bottom: 0.5em;
}

/* Looser for long-form reading */
.prose p {
  margin-bottom: 1.5em;
}
```

## Font Weight System

Standardized weight naming for design tokens:

| Token | Weight | Usage |
|-------|--------|-------|
| `--font-thin` | 100 | Decorative display only (large sizes) |
| `--font-light` | 300 | Secondary text, captions (use sparingly) |
| `--font-regular` | 400 | Body text, default (MOST COMMON) |
| `--font-medium` | 500 | Labels, buttons, emphasis within body |
| `--font-semibold` | 600 | Subheadings, card titles, nav items |
| `--font-bold` | 700 | Primary headings, CTAs |
| `--font-extrabold` | 800 | Hero headlines, display text |
| `--font-black` | 900 | Impact text, decorative (use rarely) |

**Rule of thumb:** Use a maximum of 3-4 weights per font family to keep downloads small and visual hierarchy clear. Common combinations:
- Minimal: 400 + 700
- Standard: 400 + 500 + 700
- Expressive: 400 + 600 + 700 + 900

## Font Loading Best Practices

### font-display Strategy

```css
@font-face {
  font-family: 'Inter';
  src: url('/fonts/inter-var.woff2') format('woff2');
  font-weight: 100 900;
  font-display: swap; /* Show fallback immediately, swap when loaded */
}
```

| Value | Behavior | Best For |
|-------|----------|----------|
| `swap` | Immediate fallback, swap when ready | Body text (DEFAULT) |
| `optional` | Fallback if not cached, no swap | Non-critical decorative fonts |
| `fallback` | Brief blank (100ms), then fallback, short swap window | Headings |

### Google Fonts Optimized Loading

```html
<!-- Preconnect for faster DNS/TLS -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<!-- Load only needed weights with display=swap -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### Subsetting for Self-Hosted Fonts

Reduce file size by including only needed character sets:

```bash
# Using pyftsubset (fonttools)
pyftsubset Inter-Regular.ttf \
  --output-file=Inter-Regular-Latin.woff2 \
  --flavor=woff2 \
  --layout-features='kern,liga,calt' \
  --unicodes='U+0000-00FF,U+0131,U+0152-0153,U+02BB-02BC,U+02C6,U+02DA,U+02DC,U+2000-206F,U+2074,U+20AC,U+2122,U+2191,U+2193,U+2212,U+2215,U+FEFF,U+FFFD'
```

### System Font Stacks (Zero-Download Alternative)

When web font loading is not desired:

```css
/* Modern system sans-serif */
--font-sans: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
  'Helvetica Neue', Arial, 'Noto Sans', sans-serif,
  'Apple Color Emoji', 'Segoe UI Emoji';

/* System serif */
--font-serif: 'Iowan Old Style', 'Palatino Linotype', 'URW Palladio L',
  P052, Georgia, serif;

/* System monospace */
--font-mono: ui-monospace, 'Cascadia Code', 'Source Code Pro',
  Menlo, Consolas, 'DejaVu Sans Mono', monospace;
```
